# User Personas — AI Finance Coach "พี่เงิน"
*อิงจากข้อมูลจริงใน SE-3 (Requirements) + แบบสอบถามผู้ใช้จริง 34 คน*

---

## 👩‍🎓 Persona 1 — "อิง" นักศึกษา ผู้เริ่มวางแผนการเงิน
*ตรงกับ REQ-001 · กลุ่มที่ใหญ่ที่สุดในแบบสอบถาม (>60% ของผู้ตอบ)*

|  |  |
|---|---|
| **ชื่อ-อายุ** | นางสาวอิงฟ้า ทองดี, 21 ปี |
| **อาชีพ** | นักศึกษาปี 3 สาขาบริหารธุรกิจ |
| **รายได้** | ค่าขนม 8,000 บาท/เดือน (จากผู้ปกครอง) |
| **อุปกรณ์** | iPhone 13, ใช้แอปธนาคารทุกวัน |

### 🎯 เป้าหมาย
- เก็บเงินซื้อตั๋วเครื่องบินไปเที่ยวญี่ปุ่นกับเพื่อน 20,000 บาท ภายใน 6 เดือน
- อยากรู้ว่าแต่ละเดือนใช้เงินไปกับอะไรบ้าง

### 😤 Pain Points (อิงจากแบบสอบถาม)
- **"ลืมบันทึกรายจ่าย / ไม่มีวินัย"** — เคยลองจดสมุดแล้วล้มเลิกทุกครั้ง
- ไม่รู้ว่าเงินหมดไปกับอะไร รู้ตัวอีกทีก็หมดเดือนแล้ว
- ไม่ไว้ใจแอปที่ต้องกรอกข้อมูลเยอะ รู้สึกยุ่งยาก

### 💡 สิ่งที่ต้องการจากแอป
- บันทึกรายจ่ายด้วยการสแกนสลิปได้เลย **ไม่ต้องพิมพ์เอง**
- ดู Dashboard สรุปรายเดือนได้ในหน้าเดียว
- แจ้งเตือนเมื่อใกล้หมดงบ

### 📱 วิธีที่แอปช่วยได้ (REQ-001)
> สแกนสลิปจ่ายค่ากาแฟ → ระบบจัด Category เป็น "อาหาร/เครื่องดื่ม" อัตโนมัติ → กด Confirm → เสร็จใน 3 วินาที

### 💬 Quote จากแบบสอบถามจริง
> *"บันทึก/จัดหมวดหมู่รายรับ-รายจ่ายอัตโนมัติ — นั่นแหละที่ต้องการ"*

## 👨‍💼 Persona 2 — "ปอ" พนักงานบริษัท เป้าหมายออมปลดหนี้
*ตรงกับ REQ-002 · กลุ่มอายุ 26–35 ปี พนักงานบริษัท*

|  |  |
|---|---|
| **ชื่อ-อายุ** | นายปรเมศวร์ สุขใจ, 28 ปี |
| **อาชีพ** | พนักงานบริษัทเอกชน ฝ่าย Marketing |
| **รายได้** | 22,000 บาท/เดือน |
| **อุปกรณ์** | Android Samsung, ใช้ Excel จดงบบ้าง |

### 🎯 เป้าหมาย
- ปลดหนี้บัตรเครดิต 80,000 บาท ภายใน 2 ปี
- ตั้งงบรายเดือนในแต่ละหมวด (อาหาร, เดินทาง, ช้อปปิ้ง) และห้ามเกิน

### 😤 Pain Points (อิงจากแบบสอบถาม)
- **"ไม่รู้วิธีวางแผนการออม/การลงทุน"** — เงินเดือนออกก็หมดเร็วมาก
- ตั้งงบด้วย Excel แต่ก็ลืมดูอยู่ดี ไม่มีอะไรเตือน
- กังวลเรื่อง**ความปลอดภัยของข้อมูล** บัญชีธนาคาร

### 💡 สิ่งที่ต้องการจากแอป
- ตั้ง **Budget รายหมวด** และดู **Progress Bar** ว่าใช้ไปเท่าไรแล้ว
- รับ **Push Notification** เมื่อใกล้เกินงบ 80%
- ปรึกษา AI ว่า "ถ้าอยากปลดหนี้ 80k ใน 2 ปี ต้องกันเงินเดือนละเท่าไร?"

### 📱 วิธีที่แอปช่วยได้ (REQ-002)
> ตั้งงบอาหาร 5,000 บาท/เดือน → แอปแจ้งเตือน "คุณใช้ไปแล้ว 4,200 บาท (84%) เหลือ 800 บาท!" → ช่วยหยุดคิดก่อนใช้เงินครั้งต่อไป

### 💬 Quote จากแบบสอบถามจริง
> *"อยากมีระบบคำนวณให้ว่าถ้าจะเก็บเงินเดือนละเท่าไรถึงจะพอ"*

## 🧑‍💻 Persona 3 — "บิว" ฟรีแลนซ์ รายได้ไม่แน่นอน
*ตรงกับ REQ-003 · กลุ่มฟรีแลนซ์/ธุรกิจส่วนตัว*

|  |  |
|---|---|
| **ชื่อ-อายุ** | นายจิณณพัฒน์ บุญแพง, 22 ปี |
| **อาชีพ** | ฟรีแลนซ์กราฟิกดีไซเนอร์ + ขายของออนไลน์ |
| **รายได้** | 15,000–40,000 บาท/เดือน (แปรผัน) |
| **อุปกรณ์** | MacBook + iPhone, ใช้ Notion จดงาน |

### 🎯 เป้าหมาย
- อยากรู้ว่าแต่ละเดือน "กำไร" จริงๆ เท่าไร หลังหักค่าใช้จ่ายทุกอย่าง
- อยากวางแผนเก็บเงินสำรองฉุกเฉินให้ได้ 3 เดือน (ประมาณ 50,000 บาท)

### 😤 Pain Points (อิงจากแบบสอบถาม)
- **"รายได้ไม่แน่นอน วางแผนยาก"** — บางเดือนได้เยอะ บางเดือนแทบไม่มี
- ต้องการดูภาพรวมรายได้-รายจ่ายย้อนหลังได้อย่างอิสระ ไม่จำกัดแค่รายเดือน
- กังวลเรื่องภาษีฟรีแลนซ์ ไม่รู้จะคำนวณยังไง

### 💡 สิ่งที่ต้องการจากแอป
- **Dashboard ยืดหยุ่น** ดูรายวัน/สัปดาห์/เดือน/กำหนดเองได้
- Pie Chart แยกหมวดหมู่รายจ่ายชัดเจน (ต้นทุนงาน vs ค่าครองชีพ)
- แชทถาม AI ว่า "เดือนนี้รายได้ไม่แน่นอน ควรกันเงินสำรองเท่าไรดี?"

### 📱 วิธีที่แอปช่วยได้ (REQ-003)
> เลือกดู Dashboard ย้อนหลัง 3 เดือน → เห็นว่าหมวด "อุปกรณ์/ซื้อของ" กินสัดส่วน 35% → ตัดสินใจปรับลดได้ทันที

### 💬 Quote จากแบบสอบถามจริง
> *"อยากให้มีแอปที่เป็นเพื่อนเรื่องเงินที่ไว้ใจได้"*

## 👩‍🦳 Persona 4 — "แอน" มีหนี้บัตรเครดิต ต้องการที่ปรึกษา
*ตรงกับ REQ-004 · กลุ่มอายุ 26–35 ปี มีหนี้สิน*

|  |  |
|---|---|
| **ชื่อ-อายุ** | นางสาวภัทราพร กันทะตา, 31 ปี |
| **อาชีพ** | พนักงานบริษัท ฝ่าย HR |
| **รายได้** | 27,000 บาท/เดือน |
| **อุปกรณ์** | iPhone, ใช้แอปธนาคารเป็นหลัก |

### 🎯 เป้าหมาย
- จัดการหนี้บัตรเครดิต 2 ใบรวม 150,000 บาท ให้ได้ภายใน 3 ปี
- อยากได้คำแนะนำว่าควรโปะหนี้ใบไหนก่อน ตามสภาพการเงินจริงๆ ของตัวเอง

### 😤 Pain Points (อิงจากแบบสอบถาม)
- **"มีหนี้สินและไม่รู้วิธีจัดการ"** — ลองหาข้อมูลใน Google แต่ได้คำตอบกว้างมาก ไม่ตรงกับตัวเอง
- กลัวถามคนรู้จักเพราะเรื่องเงินเป็นเรื่องส่วนตัว
- กังวลว่า AI จะแนะนำสิ่งผิดหรือไม่ปลอดภัย

### 💡 สิ่งที่ต้องการจากแอป
- พูดคุยกับ **AI Coach ภาษาไทยธรรมชาติ** ตลอด 24 ชม. โดยไม่ต้องกลัวถูกตัดสิน
- AI ต้องรู้ข้อมูลการเงิน "จริงๆ ของฉัน" ไม่ใช่ตอบแบบทั่วๆ ไป
- ต้องการ Disclaimer ชัดเจนว่า AI ไม่ใช่นักวางแผนการเงินที่มีใบอนุญาต

### 📱 วิธีที่แอปช่วยได้ (REQ-004)
> พิมพ์ถาม "พี่เงิน ฉันมีหนี้บัตร 2 ใบ ใบแรก 80k ดอก 18% ใบสอง 70k ดอก 20% ควรโปะใบไหนก่อน?" → พี่เงินดึงข้อมูลรายจ่ายจริงของเธอประกอบ → แนะนำเทคนิค Avalanche (โปะดอกสูงก่อน) พร้อม Disclaimer

### 💬 Quote จากแบบสอบถามจริง
> *"เวลาไปกินข้าวกับเพื่อนอยากให้มีระบบหาร — และอยากได้คำแนะนำด้านการเงินที่เป็นของตัวเองจริงๆ"*
